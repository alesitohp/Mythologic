Reglas del juego

    Mythologic se juega mediante el uso de dados, habiendo diferentes tipos de dados cada uno para un uso determinado.
    Dado de 4 caras:


    Usado para dar algún punto de vida o golpe entre personajes y monstruos

    Dados de 6, 8, 10 y 12 caras:



    Se suelen usar para crear las características y atacar a algunos enemigos
    Dado de 20 caras:



    Usado para el desplazamiento de personaje, aunque se puede usar para atacar y/o defender dependiendo de la situación

    ¿Cómo funciona el sistema de juego D20?



    Cada personaje o monstruo tiene seis características básicas con una puntuación cada una: fuerza, destreza, constitución, inteligencia, sabiduría y carisma, todas ellas con entre 3 y 18 puntos.

    Toda aventura se dividirá en tres pilares:

    Exploración


    Esta parte es la que implica el movimiento y viaje de los personajes descubriendo el mundo y sus mitología.

    Interacción social


    Este pilar se aplicará a las situaciones donde los personajes interactúan entre ellos y entre los personajes no jugadores que aparezcan en la aventura.

    Combate


    En este pilar se puede integrar toda lucha contra enemigos en la aventura, ya sea con las cortantes espadas y hachas, cómo lanzando hechizos o flechas.
